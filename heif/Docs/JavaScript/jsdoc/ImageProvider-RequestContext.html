Class: RequestContext

ImageProvider~ RequestContext

new RequestContext(callback, itemIds, stream, dependencies, payload)

RequestContext is used to queue data requests and is pushed to HevcDecoder.
Parameters:
Name Type Description
callback function Called when all items (frames) are decoded. The result is passed a as parameter.
itemIds Array.<number> Array of item id's to be decoded.
stream Uint8Array The data stream as Uint8Array.
dependencies Item id map of dependency/reference frames that are not to be displayed.
payload Payload The object returned to the caller.
